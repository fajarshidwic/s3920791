To run the function wrtie "./ppd ./stock.dat ./coin.dat"
To run test cases: "./ppd ./input/test1/stock.dat ./input/test1/coins.dat < ./input/test1/input.txt > ./output/test1/output.txt"

Use "make run" to run the program
