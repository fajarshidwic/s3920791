Run compiling function by typing "make ppd" in the terminal.
To run the function wrtie "./ppd ./stock.dat ./coin.dat"
To run test cases: "./ppd ./input/test1/stock.dat ./input/test1/coins.dat < ./input/test1/input.txt > ./output/test1/output.txt"

Use "make run" to run the program
To run test successfully: make sure there's no white space in forder name
Use "make testAll" to do all the test cases excluding the save and abort function.
